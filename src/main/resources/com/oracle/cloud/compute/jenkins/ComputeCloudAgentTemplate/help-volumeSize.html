A storage volume is a virtual disk that provides persistent block storage space for instances in Oracle Cloud Infrastructure Compute Classic Service.

Use one of the following abbreviations for the unit of measurement:

  • B or b for bytes
  • K or k for kilobytes
  • M or m for megabytes
  • G or g for gigabytes
  • T or t for terabytes
For example, to create a volume of size 10 gigabytes, you can specify 10G, or 10240M, or 10485760K, and so on. See the About Storage Volumes documentation for additional information.

If you omit the unit of measurement, G will be used. Additionally, you can use the abbreviations KB, MB, GB, or TB.
